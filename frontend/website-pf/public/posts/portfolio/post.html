The following are a collection of personal software development projects that are featured on my GitHub. Follow the links for each project to learn more and view the source code.

website-pf

A website and content management system. This is in fact the project that is powering prestonfrazier.net! The project consists of a front end written in ReatJS as well as a backend mainly written in Python. The backend consists of a REST api that handles retrieval of post information plus a management process that handles creation and modification of posts.
https://github.com/Prestonjf/website-pf

scorecenter

A tournament scoring and self scheduling application. Primarily written in PHP and JavaScript (using jQuery library), this project allows multiple users with role based access to create and manage teams, events and tournaments. While it is generic regarding the types of events and tournaments, it has been designed for and primarily used by Michigan Science Olympiad (A K-12 STEM education organization).
https://github.com/Prestonjf/scorecenter

sourcecode

A collection of coding challenges written in Python. Source Code was initially written as a trial event for Michigan Science Olympiad (A K-12 STEM education organization). The project contains several tests that focus on a central theme. The Python tests/files have questions that need to be solved with programming.
https://github.com/Prestonjf/sourcecode
